## gblog-admin

### 介绍
>
> 基于 vue-element-admin 完成的 Gblog 后台管理系统
> 
#### 完成功能
后面有时间在增加新功能以及优化
- [x] dashboard
- [x] 搜索文章
- [x] 标签展示/修改
- [x] 分类展示/修改
- [x] 文章审核/详情浏览
- [x] 文章详情
- [x] loading动画
- [x] 看板娘

### 安装依赖
```
npm install
```

### 启动服务
```
npm run serve
```

### 编译打包
```
npm run build
```
